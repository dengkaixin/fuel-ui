# 欢迎使用 fuel-ui-core

fuel 研发组件库

## 使用方法

### 安装

切换请求源后进行安装

```bash
# npm
npm config set registry http://192.168.21.100:8888
npm install fuel-ui-core -S

#yarn
yarn config set registry http://192.168.21.100:8888
yarn add fuel-ui-core
```

### 引入到项目中

```js
import { Flex } from 'fuel-ui-core';
```
